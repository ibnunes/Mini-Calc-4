Main Menu

1. Main Menu


From the main menu you can go to anywhere in Mini Calc. It is the first screen you'll see when you open the application and you should exit through it.


The main menu features one of the great news of this new Series of Mini Calc - a navigable menu. You can use your up and down arrows to select an item or use the shortcut provided. Each item can be associated to a submenu with related items.


For example, in the following figure, you can see that we selected the "Lists & Stat" item - we can select it by using the arrows or by pressing the key "4" - and it opened a submenu with more 5 items (the last one being "Back" so you can go back to the main menu).



The main goal of this system is to be simple and easy to use. Furthermore, it'll show you only what you must see and not every option available in a big pack of items and shortcuts.
