7.1. Stopwatch


The stopwatch is an independent system which doesn't use any resources from Mini Calc - apart the fact it is implemented inside Mini Calc. It does not use the Mini Calc memory nor any other system.


It can be stopped and returned from where it stopped after a journey throughout other systems in Mini Calc. However, it can't be run while you do other stuff - it can be only resumed its previous work.



This is the screen of the stopwatch: it is very minimalist (a major contract from its predecessor in 3rd Series). You have the timer, the number of registries, the last registry, the instructions and an area reserved to see the timings.
