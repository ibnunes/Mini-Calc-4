7.1.2. Running the stopwatch


With the stopwatch opened and reseted, we press "P" to start it. While it is running, we can mark times by pressing Enter.



If we want to pause it, we press "P" again. Both in paused and running status, you can stop the stopwatch by pressing "C". You can resume, reset or manage the timings.
