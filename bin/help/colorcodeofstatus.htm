7.1.1. Color code of status


The timer has a color code which indicates you its state. These are the colors:

  • Red: it is stopped;
  • Yellow: it is paused;
  • Green: it is running.



What does each status mean?


  • Stopped: the stopwatch isn't running at all. In this status, you can see the timings and save them. To reset the stopwatch or exit, the stopwatch must be in this status.
  • Paused: the stopwatch is running in the background, but the timer is "freezed". You CAN mark timings, but you can't do anything else. This is useful for some situations when you don't want to know the time that has passed while it's passing.
  • Running: the stopwatch is running and the current time is displayed on the screen. You can do the same things as in Paused status.
