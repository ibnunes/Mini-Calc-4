0.1. System requirements


You MUST read this before using Mini Calc!



Mini Calc is a very simples program whose auxiliary files are located in its directory. You should place it in a folder of your choice. You antivirus software may scan it and suspect it is a virus - it isn't a virus and the antivirus does say so for two reasons:

  1. Mini Calc usually isn't located in the Programs Folder, the usual folder for executable files;
  2. Mini Calc is mostly unknown.


Mini Calc works with 5 auxiliary files: those files are its memory. If you delete the files, you'll lose saved data. However, Mini Calc rebuild them from scratch only if you shut it down using the "Exit" option in the Main Menu.



Mini Calc requires:

  • Windows XP, Vista, 7, 8, 8.1 or 10;
  • 32 MB RAM;
  • 10MB hard disk.


This requirements may vary because of the memory files - the more data you store, the more disk and RAM memory it'll use. The lists are the biggest consumer of memory.


Mini Calc requires, out-of-the-box and with zero use, less than 1 MB hard disk space and less than 3 MB of RAM.
