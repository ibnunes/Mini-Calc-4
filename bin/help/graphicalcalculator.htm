4.4.1. Graphical calculator

(Aka plotter)



Open the graphic area


From the previous section, we have three functions defined: Y0, Y2 and Y3. Now we'll open the graphical calculator and see their graphics. For that, we press "G".



The color of each function is attributed automatically and it depends on the number of visible functions.



Change a function visibility


While you have the graphical calculator opened, you can manipulate the functions and the graphics are updated in real-time. To start with, we'll change the visibility. Each function can have an asterisk preceding it: this asterisk indicates that the function is visible in the plotter. To change a function visibility, select it using your up/down arrows and press "V":



As you can see, the parabola disappeared and with it the colors of the visible functions changed.



Customize the graphic area


You can change some settings of the plotter:

  • The zoom settings:
    • Minimum, maximum and scale of each axes.
  • The grid visibility;
  • The axes visibility.


To access these settings from the function editor press "Z" - it'll show you a menu ate the right side of the editor from where you can select and change each parameter. "ZStandard" sets the zoom to its default values but doesn't change the grid and axes visibility.




Save a BMP image


By pressing "S", Mini Calc will save the image into a BMP file which name is automatically generated: it'll be "mcGraph_" followed by a number enconded with the current time.
