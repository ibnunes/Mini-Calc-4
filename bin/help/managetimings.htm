7.1.3. Manage timings


With the stopwatch stopped, we're going to see the timings we took by pressing in "V".



While you're seeing the registries, you cannot save them into a file. For that, you must exit the table of registries (by pressing Esc) and then press "S". You'll be asked to give a name to the file. It'll be generated a plain text file you can use.
