3.1. Analysis of angle


You can introduce an angle in any available unit - degrees, radians or grads - by adding a suffix separated with a space. If this suffix is not added, Mini Calc will assume you want it to be in degrees. The suffixes are the following:

  • d - degrees;
  • r - radians;
  • pi - the value multiplied by pi radians;
  • g -  grads.


The value can be an expression to be evaluated as in the calculator (you cannot save data on memory, though). So, if you type "arctan(1) r", it'll assume 45º degress.
