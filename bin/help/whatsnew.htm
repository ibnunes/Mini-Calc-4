What's new in the 4th series?


Everything is new in this series. As the 5-year-old 3rd Series, this was programmed from scratch.


The navigation is easier and simpler - the menus are navigable using the up and down arrows to select the desired item. Each item can lead to a submenu with related items.


The colors used are limited to the absolute essential - only the errors, navigation items and some results are colored.


The calculator can evaluate natural expressions - if all the identifiers are known - and manage directly the memory.


The trigonometric analysis is more complete and offers a table with default values and a graphical representation of the unit circle.


The lists and statistical analysis are now connected and work together. And lists are now editable with a visual manager, offering great flexibility.


The equations are a completely new system that allows graphical representation and calculation of values through a navigable table. Both the graphical system and the table are customizable.


The random generator can now work with as many as 15 user-defined objects with a bar graph representation.


All the images produced by Mini Calc can be saved in BMP image files with the stroke of a single key.


Above all, Mini Calc is now much more powerful.
