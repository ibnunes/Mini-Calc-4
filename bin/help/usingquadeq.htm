4.2.1. Using 'quadeq'


When you use the quadratic formula tool, the resulting equation is stored in this variable. You can use it in the function editor as follows.


Open the function editor, select a function and press Enter so you edit it. Write "quadeq" and press Enter.



You'll see Mini Calc will build automatically the equation and it is ready-to-use.
