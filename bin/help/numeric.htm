2.3.1. Numeric functions


A numeric function works with the following data types:

  • rtInteger;
  • rtFloat.



cos(angle : rtFloat) : rtFloat


Returns the cosine of the angle, which is in radians. To convert from other units, see functions degtorad and gradtorad.



sin(angle : rtFloat) : rtFloat


Returns the sine of the angle, which is in radians. To convert from other units, see functions degtorad and gradtorad.



tan(angle : rtFloat) : rtFloat


Returns the tangent of the angle, which is in radians. To convert from other units, see functions degtorad and gradtorad.



arctan(angle : rtFloat) : rtFloat


Returns the arctangent of the angle, which is in radians. To convert from other units, see functions degtorad and gradtorad.



sin(angle : rtFloat) : rtFloat


Returns the sine of the angle, which is in radians. To convert from other units, see functions degtorad and gradtorad.



abs(number : rtFloat) : rtFloat


Returns the absolute value of a number, i.e., removes the signal from a number.



abs(number : rtFloat) : rtFloat


Returns the absolute value of a number, i.e., removes the signal from a number.



sqr(number : rtFloat) : rtFloat


Returns the number to the square. Equivalent to pow(number, 2).



sqrt(number : rtFloat) : rtFloat


Returns the square root of a number. Equivalent to pow(number, 0.5).



exp(exponent : rtFloat) : rtFloat


Returns the Euler number to the exponent. Equivalent to pow($e, exponent).



ln(number : rtFloat) : rtFloat


Returns the napierian logarithm of a number. Equivalent to logn(number, $e).



log(number : rtFloat) : rtFloat


Returns the logarithm of base 10 of a number. Equivalent to logn(number, 10) or log10(number).



log10(number : rtFloat) : rtFloat


Returns the logarithm of base 10 of a number. Equivalent to logn(number, 10) or log(number).



log2(number : rtFloat) : rtFloat


Returns the logarithm of base 2 of a number. Equivalent to logn(number, 2).



logn (number, base : rtFloat) : rtFloat


Returns the logarithm of a given base of a number.



frac(number : rtFloat) : rtFloat


Returns the fractional part of a number. E.g., from the number 3.4 it'll return 0.4.



int (number : rtFloat) : rtFloat


Returns the integral part of a number. E.g., from the number 3.4 it'll return 3.0.



round(number : rtFloat) : rtInteger


Rounds a number by defect (fractional part lesser than 0.5) or excess (fractional part greater than or equal to 0.5).



trunc(number : rtFloat) : rtInteger


Truncates a number, i.e., returns the integral part of a number as a rtInteger instead of a rtFloat. If you need it to be rtFloat, use the int function.



pow (base, exponent : rtFloat) : rtFloat


Raises the base to the exponent.



fact(number : rtInteger) : rtInteger


Returns the factorial of a positive or null number. If this number is greater than 255 or lesser than 0 an error will be generated.



nCk(n, k : rtInteger) : rtInteger


Returns the k-combinations of n, with k lesser than or equal to n and both positive or null and lesser than 255.



nPk(n, k : rtInteger) : rtInteger


Returns the k-permutations of n, with k lesser than or equal to n and both positive or null and lesser than 255.



rand : rtFloat


Returns a fractional pseudo-aleatory number between 0 and 1.



randint(a, b : rtInteger): rtInteger


Returns an integer pseudo-aleatory number between a (included) and b (excluded).



degtorad(angle : rtFloat) : rtFloat


Converts an angle from degrees to radians.



degtograd(angle : rtFloat) : rtFloat


Converts an angle from degrees to grads.



radtodeg(angle : rtFloat): rtFloat


Converts an angle from radians to degrees.



radtograd(angle : rtFloat): rtFloat


Converts an angle from radians to grads.



gradtodeg(angle : rtFloat): rtFloat


Converts an angle from grads to degrees.



gradtorad(angle : rtFloat): rtFloat


Converts an angle from grads to radians.
