5. Lists and Statistics


In the previous series of Mini Calc, these two features were distinct and separated. With the 4th Series, these two systems are now connected and work together.


You can create a list with a name of your choice and add up to 1000 elements each (read note 1).


Please note that the statistical system is very basic at this moment. It needs major improvements in future versions.




Notes:

(1) While implementing Mini Calc 4, it wasn't possible to create size-adaptable lists because they were generating lots of errors. Those errors are being studied so that this limitation can be abolished in the future.
