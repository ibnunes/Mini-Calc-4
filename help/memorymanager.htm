8.1. Memory Manager


The Memory Manager allows you to inspect each variable of Mini Calc and erase them. The constants are immutable, so they cannot be erased.


The structure of the Memory Manager is very simple: on the left you have a small menu with the types of variables you can manage and on the right will appear the specific managers.


For example, if you choose the "Numbers" item, you'll see all the 58 variables available with Mini Calc and navigate through them.



If you choose to reset all memory, you'll be asked if you are really sure.
