5.1.2. Using formulas


This is a special feature and is one of the reasons why you must avoid special characters in the name of the lists.


A formula is a mathematical expression which contains at least one reference to a list. It is used to automatically generate the values of a list from another one (or a set of other ones).


Create a new list and press "F" - you'll be asked to introduce the formula. There are some rules about the introduction of formulas:

  • There must be at least one reference to a list - otherwise, the formula is invalid;
  • If you reference more than one list, they must have the exact same dimension - otherwise, it'll be raised an error "Dimension mismatch".
  • The syntax of a reference if {list}.


For example, we have two lists, L1 and L2, with the same dimension and we want to multiply their values and add 2 to the result. We have to introduce {L1}*{L2}+2, then.
