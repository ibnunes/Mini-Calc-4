A bit of history...


I introduced myself in programming when I bought my first graphical calculator - a TI 84 Plus Silver Edition. With just the user manual, using a simples table of commands in its end, I learned to create tiny programs with it. Quickly I made very useful programs for math, chem and physics.


Two years later I learned in school the basic stuff about Pascal, a popular programming language created in 1971 by Niklaus Wirth. It's still my favorite language today and with it I made some useful programs.


With just three or four weeks of Pascal, I created a very tiny and unuseful calculator just for fun, and I called it Mini Calc. Without knowing it, I was creating my most beloved project of my life. Sure it wasn't, and it isn't, something great, but even not knowing why I just love this project.


In about three months I created several versions of Mini Calc in a range of 4 series:

  • The Zero Series was the initial prototype;
  • The 1st and 2nd Series introduced the main systems of Mini Calc;
  • The 3rd Series was the most mature, programmed from scratch and implementing huge improvements in usability.


Still, Mini Calc laked some power and ease-of-use. Furthermore, its calculation system was terrible: you only could introduce an operator and its operands at a time. To calculate bigger expressions, you needed to "dance" a lot with the Memory Manager (system introduced with the 3rd Series).



The 3rd Series


With this series, the introduction of numbers was very secure - if you introduced letters with the older series, the program would crash. It was possible to use internal commands, such as cancel to go back to the main menu or mem to use the active memory. You had a stopwatch, a basic system for statistical analysis (very basic!) and for trigonometry.


Mini Calc 3rd Series, version 3.1.2, Main Menu (in Portuguese)


This series was a mark of Mini Calc - it was the first near-useful calculator I made with very basic skills in programming.



The 4th Series


Almost five years have passed and my knowledge is much bigger and more mature. I've had contact with other programming languages, have learned from them and, above all, I swam deeper in the Pascal waters.


With a lot of new "guns" available, I finally decided it was time to implement the next iteration of my beloved "baby project".


So, with much less time free, I've implemented the various systems for weeks in the hope to create a truly useful and usable calculator.


The first version isn't exactly the most complete calculator, but it has the basic stuff for a calculator.


For the first time, the navigation is done with menus in which you can use the up and down arrows to select an item. From that item, a submenu can show additional options.


One can introduce complete expressions and Mini Calc can evaluate them for the very first time - now it can evaluate expressions such as "tan(degtorad(60)) * ($pi - 1)". And... it has a graphical calculator! In the first version you can't "navigate" through the functions - it isn't interactive -, but you can save a BMP image of the screen.


The user interface is much cleaner, simpler and user-friendly.
