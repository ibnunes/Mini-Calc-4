5.1.1. Managing lists and values


When you open the VLM for the first time, you'll have no lists at all. The only thing you'll have is a "<New>" button. This button will always exist since ir is one of the two possible ways to create new lists.



Press Enter - you'll be asked to introduce the name of the new list. Although Mini Calc doesn't say anything about it, avoid using other characters than the alphanumeric - A to Z and 1 to 9. Some special features rely on the use of special characters - if you happen to introduce one of those, Mini Calc can have an unpredictable behavior and crash.


In this case, we'll be very "traditional" and create the list "L1" - the name is case-insensitive.



After giving the name, we can see that the VLM created the new list L1 and is empty. If we select "---", we'll be asked for a new value. Lets insert 1:



This way, we can introduce new values and create new lists.


Lets see the instructions provided:

  • Navigation - we can use the four arrows to navigate through the lists and its values;
  • For managing values:
    • "+" adds a new value in the present index - no values are lost, they are moved forward;
    • "-" deletes the selected value - the other values are mover backward;
    • "Enter" edits the selected value - if you select this, you can't cancel the operation;
    • "S" selects the current value and asks you in which memory you want to save - since Mini Calc doesn't have write-access to the memories from mem1 to mem10, you'll be asked to save in one of the memories from 'a' to 'z'.
  • For managing lists:
    • "*" creates a new list without the need to select the "<New>" button;
    • "D" deletes the current list;
    • "N" edits the name of the list;
    • "F" inserts a formula into the list - read more in Using formulas.


As you can see, using the VLM is easy - you can do pretty much everything with a dozen of shortcuts.
