0.2. A quick tour through Mini Calc


Here you'll have a briefing about Mini Calc and its tools and systems and the way they work alone and together.



Calculator


The calculator is the basic tool of Mini Calc. It isn't a command prompt from where you can do everything! However, you are able to make powerful calculations - you can evaluate expressions, but you can also manipulate dates, strings and booleans. You can make conversions between these data types and, by using the special function is, you can mix booleans and numeric expressions to create conditionals.


You can use the memory and you have quick access to the last numeric result using the identifier ans.




Trigonometry


You can make an analysis of an angle in degres, radians or grads. Mini Calc will show you the results of several functions, such as sin (sine) or cosec, as well make a simple representation of the angle in the unit circle.


You have access to a table with default values and the identity circle.




Equations


This is a completely new feature of Mini Calc. Since we now have an expression evaluater, we can now make graphics of functions. Mini Calc has 10 functions available, from Y0 to Y9, for which you can manipulate their visibility in the graphic window. You can customize the zoom values, and grid and axes visibility. Also you can see the values of the function with a navigable table which it is also customizable - you can set the step of x and the start value of the table at any given moment.


You can calculate the roots and vertex of a quadratic function and watch its graphic - for that, you must only type quadeq and Mini Calc builds the equation from the last quadratic formula operation.


The graphic can be saved in a BMP image file.




Lists and Statistics


These two systems are now connected and are essentially only one. The basis of this part of Mini Calc is the VLM - Visual List Manager. The VLM allows you to completely manipulate the lists visually by navigation with the arrows keys and other shortcut keys. You can create and delete lists, add, append and erase values from each list and apply a formula in a list using other lists. The VLM has a series of own tools that allow easy access and manipulation for statistical analysis.


The functionality of statistical analysis is still basic. However, now you can generate a bar graph from a 2-Var Stat analysis. Furthermore, Mini Calc can calculate linear regression and build its equation automatically for the graphical calculator using the variable regeq.





Random


The biggest new from this tool is the chance to introduce up to 15 customized objects to random. When Mini Calc asks you the list to rand, write the object separated by comma - the spaces are ignored. Then you can randomize up to 400 million tries and generate a bar graph of the result which can be saved to a BMP image file.




Tools


For now, there is only one tool: the stopwatch. It is much cleaner than the one from the 3rd series and allows not only up to 100 timings registries but up to virtually infinite! Then, you can save the timings into a TXT text file. The stopwatch uses a simple 3-color code to indicate you its state.




System


The system has only one visible functionality as well: the Memory Manager. Introduced with the 3rd Series, the Memory Manager is much simpler now and lost some of its old functionalities. It was first introduced to allow the evaluation of bigger expressions. However, with the calculator of the 4th Series, that functionality is pointless. The Memory Manager now just shows how many variables are for each type and allows you to reset each one (or delete in case of lists) or reset all memory to default.



However, the system has as well the File Memory management to take care of and the Internal Commands which are embedded in the code of the different systems. These two systems are invisible to the user and are run while you do your stuff.
