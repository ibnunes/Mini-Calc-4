3.2. Unit circle


After giving an angle to analyse, Mini Calc is able to show it in the unit circle. A small menu is shown at the bottom of the window asking if you want to analyse a new angle, see the unit circle or go back to the main menu. If you select the second option, Mini Calc will build the unit circle - it'll show you something link this:



Then, you have two options:

  • Press Enter to close the unit circle, or
  • Press "S" to save the image in a BMP file and close the unit circle.


The name of the file is generated automatically - it is "mcTrig_" followed by a number encoded using the current time.
