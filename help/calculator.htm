2. Calculator


The calculator is the most basic system of all: it evaluates expressions of various types. Besides numbers, the calculator offers support for identifiers, i.e., small expressions/words that describe an object. An identifier can be a variable or a function.


A variable is an object which can be attributed with a value (as in mathematics).


In the other hand, a function is an object which receives one or more, or none, arguments and processes them to give a result.
