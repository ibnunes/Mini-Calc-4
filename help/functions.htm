2.3. Functions


A functions is called by its identifier, receives a set of arguments, if needed, and returns a result. Both arguments and result have specific data types. The general syntax is this:


identifier[(arg1, arg2, ..., argN)]


E.g. for the function trunc(3.4), trunc is the identifier, 3.4 is the argument (this function only takes one, a rtFloat) and it returns a rtInteger.
