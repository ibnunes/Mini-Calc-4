8.2. File Memory


When you open Mini Calc, you'll see it has almost everything the same way from when you left the last time. This is possible thanks to the File Memory System.


There are five memory files - some of them are updated while Mini Calc is running while others are only updated on exit (i.e., a normal exiting; clicking in the "X" button of the window you'll lose part of your data).


If the files don't exist in the beginning, don't worry - Mini Calc will create in the end of the session. So, if you're going to work in Mini Calc for a long time, proceed with this steps before working with it in the case some memory file is lost:

  1. Open Mini Calc;
  2. Press Esc.
  3. Wait 2 seconds.
  4. Open Mini Calc again.


Now you can work with the program in its full capacity.



The memory files


  • fn.mc4f - saves the functions from the function editor. It is independent from the plotter (graphical calculator).
  • par.mc4p - saves the variables of the calculator.
  • plt.mc4gd - saves the definitions of the plotter.
  • plt.mc4ge - saves the equations of the plotter.
  • vlm.mc4l - saves the lists from VLM.
