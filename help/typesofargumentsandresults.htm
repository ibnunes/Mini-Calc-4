2.1. Types of arguments and results


Each data type has a specific name used by the calculator:

  • rtInteger: describes an integer number, i.e., without fractional part;
  • rtFloat: floating point number, i.e., with fractional part;
  • rtString: alphanumeric value, i.e., words, small sentences and other things that includes letters and symbols and not only digits.
  • rtDateTime: date and time descriptor.


Note: "rt" is the acronym of "Result Type".



Each functions expects a given number of arguments, and each argument must be exactly of the data expected type.
