4.1. Function editor


The function editor is a visual tool that allows you to manipulate the 10 functions of Mini Calc. From here you can create, delete and edit functions.


What can you do with these functions?

  • See their values using the table of values;
  • Build its graphical representation using the plotter (graphical calculator).


Each of this systems is customizable in a range of definitions.



General view


This is the appearance of the function editor:



On the left side you have the editor and the right side is reserved for auxiliary operations. On the bottom-left corner you have the main instructions about the shortcuts to functionalities of the function editer. Note that not all the shortcuts are available at all times. For example, you cannot save the image of the graphic unless you have it opened.



Editing a function


Using the up and down arrows, select the function you want to edit. In this case, we'll use the first one, Y0. We press Enter and, at the bottom of the window, we are asked to give a "new eq", i.e., a new equation/function.



In this case, we'll give the equation x2/3-5. We press Enter. Since this equation is valid, Mini Calc will accept it and update Y0. If we try to introduce an invalid function or an empty one, Mini Calc will ignore and the current function is maintained.
