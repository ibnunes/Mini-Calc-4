8.4. Exception Manager


This system is not fully implemented as this first versions are yet a proof of concept. If any error occurred, this system would mask it and we would never know what went wrong.

However, it is being subtly tested in specific parts, such as the calculator, since they have suffered intense tests and their stability is confirmed.
